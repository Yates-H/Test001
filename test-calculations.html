Manual Test Instructions

To manually test the full application:

  1. Open the main calculator at http://localhost:8080/
  2. Verify that all input fields are present and functional
  3. Click "Hitung Biaya" (Calculate) button
  4. Verify that results are displayed correctly
  5. Test language switching between Indonesian and English
  6. Test QR code generation
  7. Test reset functionality
  8. Test responsive design on different screen sizes

Expected Results

Default Parameters:
- Fuel Price: 15,000 IDR/L
- Electricity Price: 1,500 IDR/kWh
- Monthly Distance: 1,500 km
- Fuel Car Consumption: 8 L/100km
- DM-i Electric Consumption: 15 kWh/100km
- DM-i Fuel Consumption: 5 L/100km
- Hybrid Ratio: 30%

Expected Calculations:
1. Fuel Car Monthly Cost: (1500/100)*8*15000 = 1,800,000 IDR
2. DM-i Electric Cost: (1500*0.7/100)*15*1500 = 236,250 IDR
3. DM-i Hybrid Cost: (1500*0.3/100)*5*15000 = 337,500 IDR
4. DM-i Total Cost: 236,250 + 337,500 = 573,750 IDR
5. Monthly Savings: 1,800,000 - 573,750 = 1,226,250 IDR
6. Savings Percentage: (1,226,250 / 1,800,000) * 100 = 68.1%
