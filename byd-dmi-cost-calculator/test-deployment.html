Deployment Steps

1 Update Configuration

Edit config.js with your deployment URL:

// config.js - Update this line
PRODUCTION_URL: 'https://your-username.github.io/byd-dmi-calculator/',
Configuration needs update

2 Push to GitHub

Upload your code to GitHub:

# Navigate to project directory
cd byd-dmi-cost-calculator

# Add all files
git add .

# Commit changes
git commit -m "Deploy for mobile access"

# Push to GitHub
git push origin main

3 Enable GitHub Pages

In your GitHub repository:

  1. Go to Settings → Pages
  2. Select main branch
  3. Select / (root) folder
  4. Click Save
Free HTTPS included
